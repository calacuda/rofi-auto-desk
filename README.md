# rofi-auto-desk
a simple python script for applying auto-desk layouts.

## Introduction
this project is a companion application to [desktop-automater](https://github.com/calacuda/desktop-automater), and, [mycroft-linux-control-skill](https://github.com/calacuda/mycroft-linux-control-skill). the later of which is a [mycroft](https://mycroft-ai.gitbook.io/docs/using-mycroft-ai/get-mycroft/linux) skill allow for voice control of a Linux system.

## Installation
```
$ git clone https://github.com/calacuda/rofi-auto-desk
$ python3 -m pip install ./rofi-auto-desk
```
